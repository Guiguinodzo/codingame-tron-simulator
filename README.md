# codingame-tron-simulator

## About

The software emulates the ["Tron Battle" game by Codingame](https://www.codingame.com/ide/puzzle/tron-battle).

Only supports Python (for now). Can easly be made to run any other program.

## Getting started

### Setup

On linux, run:
```shell
./setup_env.sh
```

On Windows, run:
```shell
./setup_env.bat
```

### Run

On linux, run:
```shell
./run.sh
```

On Windows, run:
```shell
./run.bat
```
